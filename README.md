# Udemy Homepage Clone

A pixel-perfect, responsive front-end replication of the [Udemy](https://www.udemy.com/) homepage. This project demonstrates the ability to decompose a complex, commercial-grade UI into semantic HTML components and modern CSS layouts.

### 1. Semantic HTML Structure
Built using high-standard semantic tags (`<nav>`, `<section>`, `<article>`, `<footer>`) to ensure accessibility and clear document hierarchy.

### 2. Layout Architecture
* **Flexbox:** Used for the Navbar alignment and course card distributions.
* **CSS Grid:** Utilized for the "Popular Courses" and "Recommended Topics" sections to maintain consistent spacing.
* **Responsive Design:** Implemented custom breakpoints to transition smoothly from desktop to tablet and mobile views.

### 3. Key Sections Replicated
* **Dynamic Navbar:** Features a simulated search bar, logo placement, and user utility links.
* **Categorical Menu Bar:** A secondary navigation layer mimicking Udemy's course categories.
* **Course Showcases:** * *Recommended Courses:* Focuses on image-heavy cards with pricing and badges.
    * *Popular Courses:* Organized layouts showcasing social proof (ratings/reviews).
* **Interactive Topics:** A "Recommended Topics" section including a styled input field for user suggestions.
* **Comprehensive Footer:** A multi-column link directory and internationalization placeholders.
